Claude Code 安装和模型配置

发表于2026-01-15|更新于2026-01-15|Claude DocsAI 工具编程助手
|总字数:1.6k|阅读时长:5分钟|浏览量:

Claude Code 是 Anthropic 推出的终端驻留式 AI 编程代理工具，能够完整理解代码库上下文，通过自然语言指令完成代码阅读、生成、修改、git
操作等任务。官网地址：https://code.claude.com

相比传统补全工具，它更像一个可执行多步计划的“同事”，特别适合中大型项目中的重构、特性开发、debug 与文档生成等工作流。在实际项目中可将编码效率提升
2–4 倍（视任务复杂度而定），尤其对理解遗留代码、跨文件协作场景有明显生产力红利。

本文聚焦安装、基础认证、常用配置，以及如何接入国内可稳定使用的智谱 GLM-4.7 模型，构建一套低成本、高可控的终端 AI
编程环境，为后续多模型路由、agent 协同实践打下基础。

功能使用

核心交互指令（必须掌握）

Claude Code 使用 / 开头的 slash command 与纯自然语言两种交互方式。

常用内置指令：

指令 作用 示例
/ask 提问（不修改文件） /ask 这个函数的性能瓶颈可能在哪里？
/generate 生成新文件或代码片段 /generate 创建一个用户登录的 RESTful 接口
/edit 修改已有文件 /edit 在 auth.js 中添加 JWT 刷新逻辑
/run 执行 shell 命令（需确认） /run npm test
/git git 操作 /git commit -m "feat: add login endpoint"
/model 查看&切换当前模型 /model list
/help 完整指令列表
