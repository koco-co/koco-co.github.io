准备工作

在开始之前，请确保你的电脑已经安装以下工具:

  • Node.js: brew install node
  • Git: brew install git

创建GitHub Pages仓库

  • 仓库名称: ${github用户名}.github.io
  • 公开性: Public
  • 分支: main

配置Hexo主题

推荐主题: Butterfly

  1. 下载主题
  1. 修改Hexo配置文件 _config.yml
  1. 根目录下备份主题配置文件_config.butterfly.yml, 可以自定义头像、背景等等, Ref. [Butterfly自定义配置](##八、Butterfly 自定义配置)

主题更新

常用Hexo命令

重点掌握命令: hexo g -d # 等价于 hexo generate && hexo deploy, 将 public 部署到GitHub

创建GitHub Actions自动部署

该路径下创建文件: .github/workflows/deploy.yml

区别:

  • hexo deploy: 由 hexo 将public/ 部署到远程仓库 https://github.com/${用户名}/${用户名}.github.io.git
  • git push: 由 git 整个hexo-blog项目, 但是推送过程中GitHub Actions会进一步处理, 相当于执行了hexo deploy

Butterfly自定义配置

主要改动自定义配置文件_config.butterfly.yml, 改动后需要重新执行hexo generate

自定义头像

  1. 创建img目录并放入头像文件: ~/source/img/avatar.png
  2. 编辑根目录下的~/_config.butterfly.yml
